# Roadmap

## Current version: 0.0.1 alpha
This is the initial release of Greppo. It has the following components,
- Interactive Components, `number`, `select`, `multi-select` and `draw-feature`
- Visualization Components, `line_chart` and `bar-chart`.
- Map Components, `base-layer` and `overlay-layer`.

## Current Features Roadmap
### Visualization
- Multiple dataset inputs for Charts.
- Visualizations including Pie charts and others.

### UX
- UX for Draw Feature around referencing and naming features in the User script.

### Backend
- Websocket API.
- Data file caching on initial read.
